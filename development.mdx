---
title: "Development"
description: "Run the Pandabase documentation site locally."
---

<Info>**Prerequisites**: Node.js 22+, Git</Info>

<Steps>
<Step title="Clone the repository">

```bash
git clone https://github.com/pandabase/docs.git
cd docs
```

</Step>

<Step title="Install Mintlify">

```bash
npm i -g mintlify
```

</Step>

<Step title="Run the development server">

```bash
mintlify dev
```

The docs will be available at `http://localhost:3000`.

</Step>
</Steps>

## File structure

- `docs.json` — site configuration and navigation
- `index.mdx` — home page
- `quickstart.mdx` — getting started guide
- `payouts.mdx` — payout countries, fees, and timelines
- `/developers` — API docs, changelog, webhooks, SDKs
- `/images` — images and assets
